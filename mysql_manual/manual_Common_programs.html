17 與一些常用程式一起使用MySQL

17.1 與Apache一起使用MySQL

貢獻者小節包括讓你從MySQL資料庫驗証你的用戶並且也讓你將你的日誌文件記載到一個MySQL表中的程式。見B 貢獻的程式。

你可以改變 Apache日誌格式以使它很容易被MySQL讀取，把下列行放進Apache配置文件中：

LogFormat \
        "\"%h\",%{%Y%m%d%H%M%S}t,%>s,\"%b\",\"%{Content-Type}o\",  \
        \"%U\",\"%{Referer}i\",\"%{User-Agent}i\""

在MySQL中，現在你可以這樣做：

LOAD DATA INFILE '/local/access_log' INTO TABLE table_name 
FIELDS TERMINATED BY ',' OPTIONALLY ENCLOSED BY '"' ESCAPED BY '\\'
